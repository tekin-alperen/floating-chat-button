== Installation ==

1. Go to Plugins > Add New in your WordPress admin
2. Search for "Floating Chat Button"
3. Click "Install Now" and then "Activate"
4. Go to Settings > Floating Chat to configure

Or manual installation:

1. Upload `floating-chat-button` folder to `/wp-content/plugins/`
2. Activate through WordPress Plugins menu
3. Configure under Settings > Floating Chat

= Is this plugin free? =

= Do I need a WhatsApp account? =

Yes, you need an active WhatsApp number to receive messages.

= Can I customize the message text? =

Yes, you can set any custom message that will appear when users click the button.

Yes, you can choose any color or use the default WhatsApp green.
